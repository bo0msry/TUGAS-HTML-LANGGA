# TUGAS-HTML-LANGGA
1. "<html>"
Fungsi: Sebagai root atau pembungkus seluruh halaman HTML
Contoh:
<html>
</html>

2. "<head>"
Fungsi: Menyimpan informasi tentang halaman (judul, meta, dll)
Contoh:
<head>
    <title>Judul Halaman</title>
</head>

3. "<body>"
Fungsi: Menampilkan isi utama halaman web
Contoh:
<body>
    <h1>Hello World</h1>
</body>

4. "<h1>"
Fungsi: Membuat heading/judul (h1 paling besar)
Contoh:
<h1>Ini Judul</h1>

5. "<p>"
Fungsi: Membuat paragraf
Contoh:
<p>Ini paragraf</p>

6. "<a>"
Fungsi: Membuat link/hyperlink
Contoh:
<a href="https://google.com">Klik di sini</a>

7. "<img>"
Fungsi: Menampilkan gambar
Contoh:
<img src="gambar.jpg" alt="gambar">

8. "<ul>" dan "<li>"
Fungsi: Membuat list (daftar)
Contoh:
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

9. "<div>"
Fungsi: Membuat pembagian/section dalam halaman
Contoh:
<div>
    <p>Isi div</p>
</div>

10. "<br>" dan "<hr>"
Fungsi:
- "<br>" → pindah baris
- "<hr>" → garis horizontal
Contoh:
Teks baris 1<br>
Teks baris 2
<hr>
